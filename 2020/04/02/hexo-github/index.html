hexo+github

发表于 2020-04-02 | 分类于 博客搭建
  1. 安装node.js

    node.js，下载LTS版本

  2. 安装cnpm
    • 切换到root用户
      sudo su
    • 执行命令：
      npm install -g cnpm --registry=https://registry.npm.taobao.org
      使用淘宝镜像安装
  3. 使用cnpm安装hexo
    cnpm install -g hexo-cli
  4. 创建博客
    • 创建博客文件夹
      mkdir blog
    • 进入文件夹
      cd blog
    • 创建博客
      sudo hexo init
  5. 本地访问博客
    生成博客hexo g
    启动本地服务器hexo s
    可使用hexo s -p 5000指定端口
    浏览器访问localhost:4000打开博客
  6. 新建文章
    hexo new "article-title"
  7. 部署到github
    • 在github新建仓库：cqoldby.github.io
    • 在blog目录下安装部署插件
      cnpm install --save hexo-deployer-git
    • 更改主配置文件_config.yml
    • 部署
      hexo d
  8. 更换主题
    • 下载主题
      git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
    • 更改主配置文件_config.yml
      themes: yilia
